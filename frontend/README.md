# Frontend - Visual-Search-Engine

Application Next.js (App Router, TypeScript, Tailwind CSS) consommant l'API Flask du dossier [`../backend`](../backend).

Voir le [README principal](../README.md) pour l'installation, la configuration et le guide de déploiement complet.

## Démarrage rapide

```bash
npm install
cp .env.example .env.local   # puis renseigner NEXT_PUBLIC_API_URL
npm run dev
```

Application disponible sur [http://localhost:3000](http://localhost:3000).
